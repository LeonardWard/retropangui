RetroPangUI 업데이트 내역
========================

[v0.12] 2026-06-19
------------------
squashfs+initramfs+overlayfs 부팅 아키텍처 안정화

부팅 안정화 (initramfs v0.5)
  - initramfs 메모리 주소 수정: 0x10000000 → 0x30000000 (S905X5 TrustZone 충돌 회피)
  - boot.ini 제거: C5 U-Boot는 boot.scr만 지원, boot.ini 완전 무시
  - boot.scr 자동 생성: post-image.sh에서 boot.cmd → mkimage 컴파일
  - vfat 마운트 오류 화면 출력 (2>/dev/null 제거로 디버깅 용이)
  - ext4 p2에 조기 진단 로그 (FAT32 독립 확인)
  - sync+sleep 패턴으로 워치독 전 SD 쓰기 완료 보장

[v0.11] 2026-06-17
------------------
OTA 무선 업데이트 시스템 도입
  - EmulationStation 메뉴에서 업데이트 확인 및 설치 가능
  - SHA256 무결성 검증으로 안전한 업데이트 보장
  - 이전 버전 자동 백업 (retropangui.squashfs.old)
  - 업데이트 실패 시 자동 롤백

overlayfs 기반 루트 파일시스템 적용
  - squashfs 읽기 전용 루트 (빠른 부팅, 손상 방지)
  - ext4 overlay 파티션으로 설정 영구 보존
  - initramfs를 통한 안전한 마운트 순서 보장

컨트롤러 지원 개선
  - RetroPangUI 가상 패드 P1-P4 지원
  - Twin USB Joystick 어댑터 자동 인식
  - RetroArch 포트 매핑 자동 구성

파티션 구성 변경
  - p1: boot (FAT32, 3GB) — 커널, squashfs, initramfs
  - p2: overlay (ext4, 1GB) — 설정, 저장 데이터
  - p3: share (exFAT) — ROMs, BIOS, 세이브 파일
